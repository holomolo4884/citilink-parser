## 🚀 Возможности

## 🔧 Установка

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/holomolo4884/citilink-parser.git
cd citilink-parser
```

### 2. Создайте виртуальное окружение
##### Windows:
```bash
python -m venv venv
venv\Scripts\activate
```
##### macOS/Linux:
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Установите зависимости
```bash
pip install -r requirements.txt
```

### 4. Настройте переменные окружения
```bash
cp example.env .env

nano .env
```
### ⚙️ Настройка
Файл .env — переменные окружения

### ---------- НАСТРОЙКИ SELENIUM ----------
```env
# true - браузер невидим, false - виден
HEADLESS=true
# Таймаут загрузки страницы (сек)          
PAGE_LOAD_TIMEOUT=20
# Ожидание элементов (сек)       
IMPLICITLY_WAIT=3    
```
### ---------- ЗАДЕРЖКИ ----------
```env
# Пауза между моделями (сек)
DELAY_BETWEEN_REQUESTS=1.0 
# Пауза после клика (сек)      
DELAY_AFTER_CLICK=1.5   
# Пауза после поиска (сек)         
DELAY_AFTER_SEARCH=1.5           
```

### ---------- USER-AGENT ----------
```env
USER_AGENT=Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/120.0.0.0 Safari/537.36
```

### ---------- ЛОГИРОВАНИЕ ----------
```env
# DEBUG/INFO/WARNING/ERROR
LOG_LEVEL=INFO                   
```

### ---------- МОДЕЛИ (через запятую) ----------
```env
MODELS_TO_PARSE=Gorenje ECT643SYB,Gorenje ECT641BCSC,Candy CHXC64TDB,Kuppersberg ECO 603,Kuppersberg 400,Kuppersberg ECS 635,Midea MCH-B632C,Kuppersberg ECS 624,Midea MCH-B641C,Weissgauff HV 633 BS
```

### Файл src/config.py — основные настройки
- CSS-селекторы — для парсинга характеристик
- Задержки — настройка скорости работы
- Логирование — форматы и уровни
- Сохранение — имена файлов и форматы

### 🚀 Запуск
Массовый парсинг всех моделей
```bash

```

```bash

```

# 🐛 Устранение проблем
##### Ошибка: ModuleNotFoundError: No module named 'dotenv'
```bash
pip install python-dotenv
```
##### Ошибка: WebDriver not found
```bash
pip install --upgrade webdriver-manager
```
##### Ошибка: страница не загружается
Увеличьте таймауты в .env:

```env
PAGE_LOAD_TIMEOUT=30
IMPLICITLY_WAIT=5
```
##### Ошибка: не парсятся характеристики
Обновите CSS-селекторы в src/config.py (если сайт изменил вёрстку).
